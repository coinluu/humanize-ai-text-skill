# 内容类型测试

## 测试目标

验证 Skill 是否能识别不同内容类型，并采用不同改写策略。

覆盖内容类型：

- `title`
- `comment_reply`
- `email`
- `resume`
- `sales_copy`

## 测试输入

### Prompt 1：title

```text
请帮我优化这个标题，不要标题党：

AI 工作流如何帮助企业提升效率并实现智能化升级
```

### Prompt 2：comment_reply

```text
请帮我把这条评论回复改得自然一点，不要太官方：

感谢您的反馈，我们会持续优化产品体验，为用户提供更优质的服务。
```

### Prompt 3：email

```text
请把下面邮件改得更自然、礼貌、清楚：

您好，关于本次项目推进事项，我们希望您能尽快提供相关资料，以便我们更好地开展后续工作。
```

### Prompt 4：resume

```text
请优化这条简历经历，不要编数据：

参与公司内容运营工作，负责公众号文章整理、选题协助和数据汇总。
```

### Prompt 5：sales_copy

```text
请把下面销售文案改得更有信任感，不要夸大：

我们的 AI 工作流咨询可以帮助企业快速提升效率，抓住 AI 时代机会，欢迎立即咨询。
```

## 预期行为

- `title`：更清楚、有吸引力，但不误导。
- `comment_reply`：自然、具体、不过度承诺。
- `email`：礼貌、行动明确、结构清楚。
- `resume`：突出真实职责，不新增数据、职级或成果。
- `sales_copy`：说明适合谁、解决什么问题，降低压迫感，不承诺结果。

## 通过标准

- 每个 Prompt 的内容类型判断正确或合理。
- 不同类型使用不同改写策略，而不是同一种自然化模板。
- 简历没有编造指标、成果、职位或项目规模。
- 销售文案没有承诺收益、效率比例或客户结果。
- 标题没有制造误导性悬念。
