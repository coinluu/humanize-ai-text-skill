# 平台适配测试

## 测试目标

验证 Skill 是否能根据不同平台明显调整语言目标、段落节奏、语气、开头和结尾方式。

覆盖平台：

- 小红书
- 抖音口播
- 公众号
- 朋友圈
- 商务文档

## 测试输入

### Prompt 1：小红书

```text
请把下面内容改成小红书笔记风格，短段落，有真实感，不要像广告：

AI 工作流可以帮助个人和团队提升效率，优化重复性工作流程，并更好地管理日常任务。
```

### Prompt 2：抖音口播

```text
请把下面内容改成抖音口播稿，前三秒要抓人，句子短一点，适合直接朗读：

很多团队在使用 AI 工具时，只关注单个工具的功能，却忽视了工具之间的流程衔接，因此效率提升并不明显。
```

### Prompt 3：公众号

```text
请把下面内容改成公众号观点文章的表达，逻辑清楚，不要太口水：

AI 工具正在改变内容创作方式，创作者需要学习如何把 AI 放进自己的工作流程。
```

### Prompt 4：朋友圈

```text
请把下面内容改成朋友圈分享，像真人说话，低营销感：

AI 工作流能够帮助我们提升时间管理能力，优化工作流程，实现更高效的个人成长。
```

### Prompt 5：商务文档

```text
请把下面内容改成商务文档表达，自然但专业，不要太口语：

为了提升团队协作效率，我们需要引入 AI 工作流工具，帮助项目成员更好地完成任务协同。
```

## 预期行为

- 小红书：短段落、具体、轻判断，有适合/不适合或注意点，不虚构亲身体验。
- 抖音口播：前三秒有钩子，短句，适合朗读，有停顿。
- 公众号：观点明确，解释充分，段落推进自然。
- 朋友圈：简短、自然、低压力，不像广告。
- 商务文档：专业、克制、结构清楚，不口水化。

## 通过标准

- 不同平台输出风格明显不同。
- 所有版本都保留原意。
- 没有虚构客户、数据、收入或个人经历。
- 没有为了平台化而夸大效果。
- 每个输出都符合目标平台的段落和语气规则。
