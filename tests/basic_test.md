# Basic Test

## 测试目标

验证 Skill 在普通 AI 文本自然化场景下是否能：

- 去除明显 AI 味和模板感。
- 在未指定平台时使用通用自然化策略。
- 默认输出 `standard` 格式。
- 保留原意，不新增事实、案例或数据。

## 测试输入

### Prompt 1：普通 AI 文本去 AI 味

```text
请帮我把下面这段话去 AI 味，改得自然一点：

在当今 AI 快速发展的时代，越来越多企业开始关注 AI 工作流。AI 工作流不仅能够帮助企业提升效率，而且能够优化业务流程，推动企业实现智能化升级。因此，企业应该积极拥抱 AI 工作流，以便在未来竞争中获得更大的发展空间。
```

### Prompt 2：不指定平台的自然改写

```text
这段话有点像 AI 写的，帮我改自然，但不要加新内容：

个人成长需要持续学习和不断复盘。通过系统化的方法，我们可以不断提升自身能力，并在复杂环境中实现更好的发展。
```

### Prompt 3：默认 standard 输出

```text
帮我润色一下这段 AI 味很重的内容，保留原意：

内容创作者需要充分利用 AI 工具提升内容生产效率，并通过多维度优化实现账号的长期高质量发展。
```

## 预期行为

- 自动选择 `basic_humanize`。
- 自动识别 `generic_text` 或相近内容类型。
- 默认选择 `standard` 输出模式。
- 输出应包含：
  - `AI 感诊断`
  - `改写策略`
  - `真人化改写版`
  - `修改说明`
  - `风险提示`
- 改写结果应明显减少宏大开头、模板连接词和抽象词。
- 不添加真实客户、真实数据、个人经历或具体结果。

## 通过标准

- 原文核心意思保留。
- 改写结果比原文自然，句式有变化。
- 未出现未经用户提供的案例、数据或经历。
- 未指定平台时，没有强行套用小红书、抖音或公众号风格。
- 输出结构符合 `standard` 格式。
