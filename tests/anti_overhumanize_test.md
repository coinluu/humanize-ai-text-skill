# Anti-Overhumanize Test

## 测试目标

验证 Skill 是否避免为了真人感而强行口水化、营销号化、装熟、吐槽化或不适配场景。

重点检查：

- 避免强行使用"姐妹们"、"说真的"、"我真的服了"。
- 避免把商务文本改成营销号。
- 避免过度吐槽、装熟、口水化。

## 测试输入

### Prompt 1：禁止不适配口头禅

```text
请把下面内容改得更像真人，但不要太夸张：

这款任务管理工具可以帮助用户整理待办事项，并更清楚地查看任务进度。
```

### Prompt 2：商务文本不要营销号化

```text
请把下面商务说明改自然一点：

本次流程优化将帮助团队减少重复沟通，提升任务协作效率，并进一步明确项目推进过程中的责任分工。
```

### Prompt 3：专业内容不要过度吐槽

```text
请把下面内容改得没那么官方，但仍然专业：

在项目推进过程中，信息同步不充分可能导致责任边界不清、问题响应延迟和执行效率下降。
```

## 预期行为

- 不出现"姐妹们"、"说真的"、"我真的服了"等不适配表达。
- 不出现"直接起飞"、"谁用谁知道"、"救命"等夸张营销号表达。
- 商务文本仍保持专业和清楚。
- 可以更自然，但不能变得轻浮、油腻或过度熟络。

## 通过标准

- 改写结果自然、克制、有判断。
- 没有强行个人化或虚构经历。
- 没有夸大效果。
- 商务和专业文本没有失去正式度。
- 修改说明应能体现防过度真人化的处理。
