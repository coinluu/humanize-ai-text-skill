# WeChat Group Message

我开源了一个 Agent Skill：Humanize AI Text Skill。

它主要用来把 AI 生成的中文初稿改得更自然、更像真人表达，也能适配小红书、抖音口播、公众号、朋友圈、销售文案和商务文档。

这个项目不是 SaaS，不需要数据库，核心是 prompts、rules、examples 和 tests。

重点做了几个边界：

- 保留事实
- 不编造数据
- 不虚构案例
- 参考样本只提炼通用表达规律
- 高风险内容降低改写强度

适合做 Agent、内容工作流、中文写作工具的朋友参考。
