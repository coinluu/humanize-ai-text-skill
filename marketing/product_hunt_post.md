# Product Hunt Post

## Tagline

Agent-ready skill for humanizing AI-generated Chinese text.

## Description

Humanize AI Text Skill helps agents rewrite AI-generated Chinese drafts into natural, human-like, platform-ready content.

It supports:

- Basic AI-tone reduction.
- Platform adaptation for Xiaohongshu, Douyin, WeChat, Moments, sales copy, and business documents.
- Reference sample style extraction.
- Protected facts preservation.
- Safety checks.
- Anti-overhumanization.

It is open source, stateless, and does not require a database or backend.

## First Comment

I built this because many Chinese AI-generated drafts are structurally correct but still feel stiff, generic, or platform-mismatched.

This project packages the rewrite workflow as an Agent Skill: prompts, rules, examples, and tests.

It focuses on naturalness, factual consistency, platform fit, and responsible rewriting.
