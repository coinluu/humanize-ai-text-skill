# 朋友圈示例

## 场景

朋友圈内容。要求像真人分享，低营销感，不像广告。

## 输入信息

```yaml
mode: platform_humanize
platform: moments
content_type: caption
rewrite_intensity: Level 2
output_mode: standard
allow_hypothetical_scenarios: false
```

## 改写前

```text
AI 工作流是提升个人效率的重要工具，可以帮助我们更好地管理时间、优化流程、释放个人潜能。欢迎大家一起学习 AI 工作流，开启高效成长之路。
```

## AI 感诊断

- "重要工具、优化流程、释放潜能"偏宣传。
- "欢迎大家一起学习"像课程广告。
- 语气过满，不像朋友圈自然分享。
- 缺少真实的轻判断和日常表达。

## 改写策略

- 降低营销感，改成个人观察式表达。
- 保留 AI 工作流和效率主题。
- 不伪造个人经历，不写具体成果。
- 使用短句，让语气更像真实朋友圈。

## 改写后

```text
最近越来越觉得，AI 工作流最有用的地方，不是让人显得更忙，而是把一些重复的小事先理顺。

时间省下来以后，反而更容易看清楚：哪些事该自己判断，哪些事其实可以交给工具先处理。
```

## 修改说明

- 删除广告式邀请和成长口号。
- 把"提升效率"改成更自然的个人观察。
- 保留"AI 工作流帮助处理重复事务"的核心意思。
- 语气更轻，不强行推销。

## 风险提示

未发现需要特别提示的风险。没有加入具体个人经历或效果承诺。
