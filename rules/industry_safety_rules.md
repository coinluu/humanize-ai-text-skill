# Industry Safety Rules

## 规则说明

高风险行业或敏感场景中，文本自然化必须优先保证准确、安全和边界。改写不应增强未经证实的承诺，也不应删除必要限定词。

## 适用场景

- 文本涉及医疗健康、法律、金融投资、教育培训、求职简历、情感关系、亲子育儿、商业收益或本地生活真实案例。
- 用户要求增强说服力，但原文包含可能影响决策的结果、收益、疗效、录取、法律或安全判断。
- 文本包含客户反馈、个人经历、权威背书、案例数据等需要核验的信息。

## 禁止事项

- 禁止承诺结果。
- 禁止虚构案例。
- 禁止虚构数据。
- 禁止夸大收益、疗效、录取、法律结果或求职结果。
- 禁止制造焦虑。
- 禁止伪造经历、身份、认证或权威背书。
- 禁止删除必要限定词、风险提示和免责声明。

## 通用安全规则

- 不得承诺结果。
- 不得虚构案例。
- 不得虚构数据。
- 不得夸大收益。
- 不得给确定性法律、医疗、金融结论。
- 不得制造焦虑。
- 不得伪造经历。
- 不得伪造权威背书。
- 必须保留必要限定词。
- 默认降低改写强度到 Level 1-2。

## 医疗健康

适用场景：

- 疾病、治疗、药品、营养、心理健康、医美、康复。

禁止事项：

- 承诺疗效。
- 替代医生诊断。
- 删除"可能"、"建议咨询医生"等限定。

示例：

```text
错误：这套方法能治好失眠。
正确：这套方法可能有助于改善睡眠习惯，但如果长期失眠，仍建议咨询专业医生。
```

## 法律

适用场景：

- 合同、纠纷、诉讼、合规、责任、劳动争议。

禁止事项：

- 给确定性判决结论。
- 删除适用条件。
- 把一般信息写成法律意见。

示例：

```text
错误：这种情况一定能胜诉。
正确：这种情况需要结合证据、合同条款和当地法律判断，不能仅凭描述下结论。
```

## 金融投资

适用场景：

- 投资、理财、保险、贷款、税务、副业收益。

禁止事项：

- 承诺收益。
- 暗示稳赚。
- 虚构历史收益或用户结果。

示例：

```text
错误：跟着做就能稳定赚钱。
正确：这类方法存在风险，是否适合要看个人资金情况、风险承受能力和具体执行。
```

## 教育培训

适用场景：

- 升学、考试、课程、训练营、学习效果。

禁止事项：

- 承诺提分、录取、通过率。
- 制造家长或学生焦虑。

示例：

```text
错误：报名后孩子一定能提高成绩。
正确：课程可以提供学习方法和练习支持，但效果仍取决于基础、投入时间和执行情况。
```

## 求职简历

适用场景：

- 简历、面试、项目经历、薪资、职位申请。

禁止事项：

- 伪造公司、职位、职责、项目、数据、奖项。
- 把参与写成负责，把辅助写成主导。

示例：

```text
错误：主导千万级项目增长。
正确：参与项目执行，负责其中的数据整理和用户反馈分析。
```

## 情感关系

适用场景：

- 恋爱、婚姻、分手、家庭关系、人际冲突。

禁止事项：

- 制造对立和焦虑。
- 给绝对判断。

示例：

```text
错误：他这样就是不爱你。
正确：这个行为可能说明沟通方式存在问题，但还需要结合你们的具体相处情况判断。
```

## 亲子育儿

适用场景：

- 儿童教育、亲子关系、行为管理、成长建议。

禁止事项：

- 制造父母焦虑。
- 给极端化判断。
- 把个别经验当普遍结论。

示例：

```text
错误：不这样做孩子以后一定会落后。
正确：这种做法可以作为一种参考，但孩子的发展节奏不同，需要结合具体情况调整。
```

## 商业收益承诺

适用场景：

- 招商、加盟、课程、咨询、副业、商业项目。

禁止事项：

- 保证收益。
- 虚构成功案例。
- 淡化投入、风险和条件。

示例：

```text
错误：普通人也能月入过万。
正确：项目收益受资源、执行、市场和投入影响，需要谨慎评估。
```

## 本地生活真实案例

适用场景：

- 餐饮、门店、探店、客户评价、服务体验。

禁止事项：

- 虚构到店体验。
- 虚构客户评价。
- 虚构排队、销量、口碑。

示例：

```text
错误：我上周去吃了，真的很多人排队。
正确：如果要表达受欢迎，可以基于用户提供的真实信息写；没有真实体验时不要写成亲历。
```

## 示例

Source:

```text
这套训练营可以帮助学员梳理学习计划。
```

Wrong:

```text
参加训练营后，学员基本都能明显提分。
```

Correct:

```text
这套训练营主要帮助学员梳理学习计划和执行节奏。实际效果仍取决于基础、投入时间和完成情况。
```
