# Anti-Overhumanize Rules

## 规则说明

防止 Agent 为了追求真人感，把文本改得过度口语、低俗、油腻、夸张或不符合场景。高级真人感应该自然、克制、有判断、有细节、有取舍、有节奏，而不是表演式表达。

## 适用场景

- 用户要求"更像真人"、"更有网感"、"别那么 AI"。
- 文本属于商务、法律、医疗、教育、求职、专业解释等需要稳重表达的场景。
- 平台内容需要自然化但不能变成低质营销号。

## 禁止事项

- 禁止强行口水化。
- 禁止强行加入"说真的"、"我真的服了"、"姐妹们"等不适配表达。
- 禁止强行制造情绪。
- 禁止强行制造个人经历。
- 禁止强行装熟。
- 禁止把专业内容改成低质营销号。
- 禁止把商务内容改成过度口语化内容。
- 禁止把严肃内容改成轻浮表达。
- 禁止用夸张标点、过度感叹、强刺激词制造热闹。

## 正确方向

- 用清楚判断代替空泛口号。
- 用自然停顿代替满篇连接词。
- 用具体动作代替抽象大词。
- 用适度语气代替强行亲密。
- 用边界意识代替绝对化承诺。

## 示例

## 错误示例

Before:

```text
本方案可以帮助团队提升沟通效率。
```

Bad After:

```text
说真的，这个方案我真的服了，团队沟通直接起飞，谁用谁知道！
```

问题：

- 语气不适配商务场景。
- 夸大效果。
- 加入无来源的体验感。

## 正确示例

After:

```text
这套方案主要解决团队沟通里信息分散、跟进不清的问题。流程固定下来后，谁负责、进展到哪一步，会更容易看清楚。
```

## 高风险示例

Before:

```text
这套课程能帮助学生提升备考效率。
```

Bad After:

```text
报了这套课，提分真的会快很多，家长完全不用焦虑。
```

Correct After:

```text
这套课程的重点，是帮助学生把复习节奏和错题整理做得更清楚。它可以作为备考辅助，但学习效果仍然取决于基础、投入时间和执行情况。
```
