# 参考样本风格提炼规则

## 规则说明

参考样本只能用于提炼通用表达规律，最终改写必须基于用户原文重新组织表达。样本不是事实来源，也不是可复用素材库。

## 适用场景

- 用户提供参考文章、参考文案、参考口播稿或参考标题。
- 用户要求"参考这个风格"、"学一下这种表达"、"按这个调性改"。

## 允许提炼

- 开头方式
- 段落节奏
- 句式长短
- 语气强度
- 观点表达方式
- 内容推进顺序
- 读者沟通方式
- 标题构造逻辑
- 结尾收束方式

## 不作为可迁移素材

- 样本中的具体故事
- 样本中的个人经历
- 样本中的具体案例
- 样本中的独特比喻
- 样本中的原句表达
- 样本中的专属标题
- 样本中的强识别性表达
- 样本中的人物、品牌、地名、事件和数据

## 禁止事项

- 禁止复用样本原句、标题、独特比喻或强识别性表达。
- 禁止把样本中的个人经历写进用户文本。
- 禁止把样本中的案例、数据、人物或品牌当成用户事实。
- 禁止让样本风格覆盖用户原文的事实边界和使用场景。

## 执行规则

- 先分析样本的表达规律，再回到用户原文。
- 把样本特征转成抽象指导，例如"短段落推进"、"先判断后解释"。
- 不从样本中借用事实、故事、案例或语句。
- 如果样本风格与用户场景不适配，以用户场景为准。
- 如果样本语气夸张或风险较高，降低迁移强度。

## 示例

参考样本特征：

```text
样本常用短段落，先给判断，再解释为什么。
```

允许迁移：

```text
改写时可以先给出核心判断，再用两三个短段解释原因。
```

禁止迁移：

```text
直接复用样本中的开头故事、标题、比喻或原句。
```

## 正确示例

User 原文：

```text
很多人在学习新技能时缺少持续反馈，因此难以坚持。
```

参考规律：

```text
短段落；先判断；再解释原因；结尾给出克制建议。
```

改写：

```text
学新技能最难的，往往不是开始，而是坚持到看见反馈之前。

如果每一步都太大，人很容易觉得自己没进展。更稳的做法，是把目标拆小，让每次练习后都能看到一点变化。

反馈变清楚了，学习才更容易持续。
```
